# Личный сайт-визитка (минимализм)

Стек: **HTML + CSS + JS**, без сборки. Готов к деплою на GitHub Pages.

## Структура
```
/
├─ index.html
├─ assets/
│  ├─ styles.css
│  └─ main.js
├─ .github/workflows/deploy.yml   # авто-деплой на GitHub Pages
├─ robots.txt
└─ sitemap.xml
```

## Локально
Просто откройте `index.html` в браузере.
